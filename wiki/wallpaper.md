wallpaper
=========

## Use feh

    $ sudo apt install feh
    $ feh --bg-scale ~/Pictures/config/wallpaper/wallpaper.jpg

## Use nitrogen

    $ sudo apt install nitrogen
    $ nitrogen <image-dir>
    $ nitrogen --restore

## Config nitrogen

    ##Select "zoom-all fill" mode
    $ cat .config/nitrogen/bg-saved.cfg
    [xin_-1]
    file=campos-de-castilla.jpg
    mode=5
    bgcolor=#000000
